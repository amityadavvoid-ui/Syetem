## 2024-05-24 - [Replace local rAF closures with a centralized loop]
**Learning:** The previous implementation of the particle system in `ui.js` created a new closure and a separate `requestAnimationFrame` loop for every single particle spawned. For visual effects like particles that can have multiple active instances simultaneously, maintaining an individual `rAF` loop per particle adds unnecessary overhead, degrades performance, and can cause visual stuttering.
**Action:** Replace multiple individual `requestAnimationFrame` closures with a single, centralized `requestAnimationFrame` loop. The spawned instances should instead be pushed to a state array containing their variables, and the central loop should iterate through that array to update the instances or filter out completed ones.
